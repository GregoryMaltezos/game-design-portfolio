Gameplay Systems

Echoes is built around interconnected systems where preparation, combat, loot collection, extraction, and progression all feed into each other. The goal is to make every run feel dangerous, valuable, and personally transformative.

Core Loop

Prepare → Enter → Fight → Loot → Extract → Upgrade → Repeat.

Each run asks the player to balance safety, greed, progression, and survival.
