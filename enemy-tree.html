Enemy Hierarchy

This hierarchy shows how enemy types are organized across common threats, elite encounters, minibosses, Uber Bosses, and late-game player-specific Echo challenges. 

Enemies
├─ Common Enemies
│  ├─ Shamblers
│  ├─ Skulkers
│  ├─ Spinecasters
│  └─ Thrash Beasts
│
├─ Elite Enemies
│  ├─ Guardian Wraiths
│  ├─ Corrupted Brutes
│  └─ Shrieking Nodes
│
├─ Minibosses
│  ├─ Thornbound Matriarch
│  ├─ Ashbound Goliath
│  └─ Hollow Prophet
│
├─ Uber Boss
│  ├─ Summoned through a mini quest
│  ├─ One summon item per run
│  ├─ Cannot be summoned in confined spaces
│  └─ Defeating it opens instant extraction and grants high-tier loot
│
└─ Player-Specific Echo Bosses
   ├─ Ultimate Challenge
   │  ├─ Triggered at maximum Echo attunement
   │  └─ Defeating it allows Echo reset
   │
   ├─ Echo Legacy Duel
   │  ├─ Challenge the previous wielder's Echo
   │  └─ Cosmetic reward: "[Archetype] Master"
   │
   ├─ Prestige / Rebirth Trial
   │  ├─ Follows the Legacy Duel
   │  ├─ Cosmetic-only reward: "True [Archetype] Master"
   │  └─ Locks archetype for future runs
   │
   └─ True Jack of All Trades
      ├─ Achieved by mastering all archetypes
      └─ Cosmetic recognition with limited cross-archetype versatility
